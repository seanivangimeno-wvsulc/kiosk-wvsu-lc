# Project Charter: WVSU-LC Kiosk

## Project Title
West Visayas State University - Lambunao Campus (WVSU-LC) Interactive Information Kiosk

## Executive Summary
WVSU-LC requires a modern, interactive, and robust information kiosk system to serve as a digital concierge for students, faculty, staff, and campus visitors. This system will be deployed on physical touch screen kiosks around the campus.

## Project Objectives
- **Modernize Campus Navigation**: Replace static directories with an interactive SVG campus map.
- **Improve Communication**: Centralize school announcements, bulletins, and event calendars on a live-updating dashboard.
- **Optimize Resource Lookup**: Provide quick directory search for academic departments, faculty offices, and support facilities.
- **Ensure Digital Inclusion**: Build with accessibility in mind (adjustable text sizes, high-contrast themes, speech assist options).

## Stakeholders
- **Project Sponsor**: WVSU-LC Campus Administration
- **Project Manager**: Sean Ivan Gimeno
- **Primary Users**: WVSU Students, Faculty, and Campus Visitors
- **IT Support Team**: System administrators responsible for kiosk hardware and server maintenance.
