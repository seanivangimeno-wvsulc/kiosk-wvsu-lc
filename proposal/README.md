# WVSU-LC Kiosk Project Proposal

This directory contains resources, documents, and research data representing the official proposal for the West Visayas State University - Lambunao Campus Interactive Kiosk.

## Project Summary
The WVSU-LC Kiosk is a touch-based interactive application designed for deployment on physical terminals throughout the Lambunao Campus. It aims to improve campus navigation, disseminate timely notices, and streamline directory lookups for students, staff, and visitors.

## Key Objectives
1. **Interactive Campus Directory**: Allow users to look up building locations, classroom numbers, and administrative staff offices.
2. **Real-time News & Notices**: Streamline communication by showing current notices and events.
3. **Accessibility**: Ensure the interface is easily accessible to all users, with large text modes and clean navigation routes.
