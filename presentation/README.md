# WVSU-LC Kiosk Presentation Materials

This directory contains slides, demonstration notes, scripts, and video walkthroughs of the WVSU-LC Kiosk project.

## Deliverables
- **Pitch Deck**: Slides outlining the problem statement, solution design, technical stack, and future rollout roadmap.
- **Demonstration Script**: Steps and key interactions to showcase during user or stakeholder presentations.
- **Recording Assets**: Video screen recordings showing responsiveness, search functionality, and map interactions.
