# WVSU-LC Kiosk Prototype

This directory houses the prototyping assets, mockups, and early designs for the WVSU-LC Kiosk interface.

## Contents
- **Mockups/**: Design files, export images, and wireframes.
- **Figma Links**: (To be updated as design proceeds)

## Prototyping Phase Objectives

2. Optimize search interface for faculty directory.
3. Test touch UI layout sizing and responsiveness in standard landscape orientations.
