# Prompts Library: WVSU-LC Kiosk

This file catalogues helpful prompts used for code generation, designing UI components, and testing functionalities of the WVSU-LC Kiosk.

## UI/UX Design Prompts
### Campus Map Page
> "Generate a React component for an interactive campus map using SVG coordinates. Style it with custom tooltips, zoom/pan controls, and color-coded buildings matching the WVSU branding (Navy Blue & Gold). Ensure it has smooth CSS transitions and a responsive sidebar listing departments."

### Interactive Directory Search
> "Create a high-performance search component in React that filters students, faculty, and administrative offices. Implement debouncing, keyboard navigation, and custom card overlays to show office hours and contact info."

## Code Refactoring & Testing
### Touch Input Optimization
> "Review this component for touch accessibility on physical kiosk screens. Increase touch target sizes to at least 48x48px, add visual active states for touch triggers, and prevent double-tap zoom issues."
